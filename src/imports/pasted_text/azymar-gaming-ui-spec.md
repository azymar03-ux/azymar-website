# AZYMAR – COMPLETE FIGMA AI PROMPT

Create a modern, minimal, premium-quality gaming website UI named **AZYMAR** inspired by PlayHop, Poki, and CrazyGames, but with an original design. Use a clean white background, teal (#11B5A4) primary color, light gray secondary backgrounds, and dark gray typography (#1F2937). Use the Inter font family, 16px rounded corners, soft shadows, smooth hover animations, and generous spacing. The design should feel fast, elegant, and user-friendly.
